En este ejemplo, se utilizan variables CSS para definir colores y tamaños de fuente de manera consistente en todo el documento.

Se definen variables para el color de fondo, el color del texto y el tamaño de fuente, y luego se aplican a los elementos del documento.

Esto permite cambiar fácilmente los estilos de todo el documento modificando las variables en un solo lugar.
